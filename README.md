 Employee Management System

A modern Employee Management System built using a clean UI design inspired from Figma, developed with React (Vite) and modular components. This project helps manage employee data efficiently with an intuitive dashboard, filtering system, and interactive UI.

 Project Overview

This project is designed to simplify employee data management in organizations. It provides features like adding, viewing, updating, and deleting employee records with a responsive and user-friendly interface.

The UI is implemented based on a Figma design reference, ensuring a professional and modern look.

 Tech Stack
 React (Vite)
 HTML5 + CSS3
 JavaScript (ES6+)
 Component-based Architecture
 REST API integration (if backend connected)
 Figma (UI Design Reference)
 Features
 Add new employees
 View employee list in table format
 Search and filter employees
 Edit employee details
 Delete employee records
 Dashboard for overview
 Responsive UI (Mobile + Desktop)
 Clean and modern UI inspired from Figma design
